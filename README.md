# Corapak Warehouse Tools

A collection of internal warehouse utilities developed for Corapak to improve warehouse efficiency, planning, and visualization.

## Current Applications

### 📦 Pallet Configurator
Interactive pallet planning tool capable of generating multiple palletization layouts, visualizing each layout in 2D and 3D, and allowing manual pallet design.

---

## Planned Warehouse Tools

- Truck Load Estimator
- Box Fit Calculator
- Label Generator
- Carton Dimension Calculator
- Shipping Utility Tools
