# Product Vision

Build a browser-based pallet planning workspace for Corapak that combines automatic layout generation, interactive editing, accurate pallet calculations, layer visualization, and future operational integrations.

The application should remain:

- Easy for warehouse staff to use
- Free to operate with the current self-hosted setup
- Modular and maintainable
- Accurate and transparent about its calculations
- Expandable into additional warehouse tools
- Usable on desktop first, with mobile support later

The Pallet Configurator should behave more like a lightweight design workspace than a basic pallet calculator.

Long-term capabilities may include:

- Automatic pallet layout generation
- Multiple layout comparisons
- Per-layer visualization
- Interactive 3D visualization
- Free-design mode
- Dragging and rotating cartons
- Layout validation
- PDF and image export
- Saved pallet patterns
- SKU and product libraries
- Odoo integration
